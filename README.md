# Predicting Disease Outcomes from Remote Monitoring using Machine Learning

This repository contains the code and project files used to generate the findings for the published systematic review:

- Wolber J, Schuppert A, Muecke M, Sellin J, et al. *Predicting disease outcomes from remote monitoring using machine learning: a systematic review*. **BMC Medical Informatics and Decision Making** (2026).  
  DOI: [10.1186/s12911-026-03495-0](https://doi.org/10.1186/s12911-026-03495-0)  
  Article: [Springer Link](https://link.springer.com/article/10.1186/s12911-026-03495-0)

## Repository contents

- `main.ipynb`: Main analysis notebook used for data processing and statistical analyses.
- `studies.csv`: Structured study-level data used in the review.
- `RoB_table.csv`: Risk of Bias (PROBAST-based) assessment table.
- `PROBAST_20190515.pdf`: PROBAST reference document.

## Notes

- This repository is intended to support reproducibility and transparency of the review findings.
- File names and outputs correspond to the analyses described in the published manuscript.
